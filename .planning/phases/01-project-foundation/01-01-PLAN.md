1. Create `src/naz/cli.py` with this exact content (per D-08 Typer, per D-09 scan command):

```python
"""Naz CLI - Security scanning for solo developers."""

import typer

app = typer.Typer(
    name="naz",
    help="Security scanning for solo developers.",
    no_args_is_help=True,
)


@app.command()
def scan(
    path: str = typer.Argument(".", help="Path to the repository to scan"),
) -> None:
    """Scan a repository and detect its technology stack."""
    typer.echo(f"Scanning: {path}")
```

3. Create venv and install: `uv venv && uv pip install -e ".[dev]"` (if dependency-groups not supported by uv pip, use `uv pip install -e .` then `uv pip install pytest pytest-cov ruff`)
